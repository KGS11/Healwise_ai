# HealWise AI

Smart Multilingual Naturopathy Healthcare and Wellness Platform.

HealWise AI is a Flutter Android mobile application for preventive healthcare,
naturopathy awareness, AI wellness guidance, medical report understanding,
yoga assistance, and educational analytics.

## Current Setup

This project is intentionally lightweight.

- Flutter frontend
- Android support enabled
- Windows desktop support not enabled
- Web support can be added later
- Firebase, AI, OCR, maps, and camera packages will be added phase by phase

## First Files To Learn

- `lib/main.dart` starts the app.
- `lib/app/healwise_app.dart` configures the app.
- `lib/features/language/language_selection_screen.dart` is the first screen.
- `lib/features/home/home_dashboard_screen.dart` is the main dashboard.
- `lib/core/theme/app_theme.dart` controls colors and UI style.

## Development Phases

1. Authentication, language selection, and home dashboard.
2. AI chatbot, report upload, and OCR analysis.
3. Yoga posture detection, audio guidance, and video guidance.
4. Progress analytics, mathematical dashboard, and DAA dashboard.
5. AI optimization, testing, Firebase deployment, and backend deployment.
